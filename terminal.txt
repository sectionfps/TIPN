------------- Sincronizacao:

Codespace > repositório
git add -A
git commit -m "update"
git push origin main

Repositorio > codespace
git fetch origin
git pull origin main

------------- Conferir branch:
git branch

------------- Puxar fluxo normal:
git pull origin main

------------- Sincronizacao forcada (CUIDADO!):

Codespace > repositorio
git add -A
git commit -m "update"
git push -f origin main

Repositorio > codespace
git fetch origin
git reset --hard origin/main
